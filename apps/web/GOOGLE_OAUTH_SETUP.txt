========================================
GOOGLE OAUTH SETUP - FOREST WEB APP
========================================

Quick Setup (3 steps):

1. GET GOOGLE CLIENT ID
   - Go to: https://console.cloud.google.com/
   - Create/Select project
   - Enable Google+ API
   - Create OAuth 2.0 Client ID (Web application)
   - Add authorized origins:
     * http://localhost:3000 (development)
     * https://yourdomain.com (production)
   - Copy Client ID

2. ADD TO ENVIRONMENT
   Create/Update .env.local file in apps/web/:
   
   NEXT_PUBLIC_API_URL=http://localhost:3001/api
   NEXT_PUBLIC_GOOGLE_CLIENT_ID=your-client-id.apps.googleusercontent.com

3. INSTALL & RUN
   cd apps/web
   pnpm install
   pnpm dev

TROUBLESHOOTING:
- If button doesn't appear: Check NEXT_PUBLIC_GOOGLE_CLIENT_ID in .env.local
- If "Invalid Client ID": Verify Client ID matches Google Cloud Console
- If CORS error: Add your domain to authorized origins in Google Console
- If network error: Verify backend is running on http://localhost:3001
